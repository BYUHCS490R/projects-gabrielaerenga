When do you use a table?

When dealing with data or statistics to place into rows or columns

When do you not use a table?

To layout elements on a web page

When do you need to do extra work for table accessibility?

You need to make sure the labels are "th" instead of "td" for the table headings. Also to help, is seperating the different sections of the table by using the "thead", "tbody", and "tfoot" tags.

Why don't we use the border attribute?

It is considered obsolete and also that it lacks customization when using CSS.

What is the easiest way to color every other row a different color?

Using the pseudo tag like tr:nth-child(even) and change color accordingly.
