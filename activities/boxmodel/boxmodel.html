Box Models

Questions:

What is the difference between block tags and inline tags?

Block tags are the default tags used in html such as header, h1, and nav tags. Block tags create boxes inside the wepage and takes up "space". Inline tags are tags that are used in the code and does not affect the overall structure of the program. For example tags like span, a, strong, em, mark can go inside block tags and would not affect structure of code.

What is the margin, border, and padding?

Padding pushes the box tag structures inside. Margin pushes the box edges out. Border creates a border around the box tag structures.

What are units of measurement you can use in CSS?

We can use px, percentages, em, rem, or pt units of measurements in CSS.

When would you use a figure tag?

Figure tag is another block tag. You can contain an image tag with a figure tag. Because an img tag is an inline tag you can give it structure with the figure tag and also add text for a caption.
