Questions

Please describe the parent-child relationship in HTML and why it becomes essential when using flex and grid.

The parent-child relationship is essential in HTML as it affects the positioning of the elements within the block tags. For instance, the parent of an html document would be the body tag. The tags within the body tags would then be considered the children. When using grid, you'll be bale to control the child tags within the parent tag.

What are the two main differences between CSS Grid and CSS Flex?

Flex is designed for 1-dimensional layouts while Grid is designed for 2-dimensional layouts.
