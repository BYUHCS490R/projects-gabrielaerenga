Questions

What is normal flow in HTML?

Normal flow is when the block tags take the full width of the screen on an HTML page.

What does float do? Why do you need to clear the float?

Float makes it so that your images can be inline with text on a web page. It helps the text float around the image. You need to clear the float to be able to seperate and posiiton the sibling tags on an HTML document.

What do absolute, fixed, and relative do?

Absolute removes an element from the normal flow. It no longer takes up space and it can overlap other elements. Fixed places an image or feature onto the window/screen of the device you are using to view the HTML page. Relative positions an element relative to its normal position in the document flow.

How can you make nice navigation buttons for mobile devices?

You can use the nav block tags with the display, padding and color functions.
