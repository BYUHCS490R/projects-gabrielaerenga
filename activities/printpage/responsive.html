Why is the viewport meta tag important?

It's important so it adjusts the web page according to the size of the screen or web browser we are viewing the page on. rather than zooming out like old versions of other web browsers

How can you adjust the CSS for printing?

We can use a media query within the link tags at the top

When would you want to adjust the CSS for printing?

Only when we know that people will print our Page
