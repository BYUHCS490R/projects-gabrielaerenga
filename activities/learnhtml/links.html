2. What is the root? What is the relationship between folder and file names and the path?

The root folder is the container for your website. In it is where you will find all the html files (and the resources that are linked in them) relative to the root folder so that your browser can find them.

3. What is a default document?

The default document is the home page of your website. You can only have 1 default document and that is an "index.html" file

4. Explain the difference between an absolute link and a relative link. Why would you use relative links?

An absolute link contains the full URL address of a specific resource, and it follows protocol like "http" or "https". A relative link is a file in relation to the location of the current html file, it only contains the path of a file.
