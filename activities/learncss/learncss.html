Beginning CSS

1. Explain the difference between inline, embedded, and external CSS.

  • inline CSS applies styles directly to a specific HTML element within the opening tag
  • embedded CSS is placed within a style tag in the head section of an HTML document
  • external CSS involves creating a seperate CSS file that contains all the style rules, it is then linked to the HTML file using the link tag in the head section of the HTML file.

2. Explain why you should only use external CSS.

External CSS is helpful as it seperates content from presentation, readability (decreased clutter in html files), and easier to update styles globally.

3. What is a selector used for?

A selector is used to select and target specific HTML elements within a web page so that CSS styles can be applied to them.
