What is the difference between an ID and a Class?

ID is naming a specific tag that cannot be duplicated with any other tag in an HTML document. Class is putting multiple tags under the same constants with multiple attributes. Classes are far more flexible than ID.

What is the difference between a SPAN tag and a DIV tag?

SPAN and DIV tag is a way to select code or text in an HTML document. DIV specifically is used for blocks of code whereas SPAN is typicaly for a line or specific texts.

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

DIV tags are non-semantic tags, meaning that they are specifically used for styling purposes. Meanwhile HEADER, MAIN and FOOTER tags have a specific meaning and purpose in an HTML document.
