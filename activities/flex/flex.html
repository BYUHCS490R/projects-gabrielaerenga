CSS Flex

What did you learn about Flex and CSS?

I learned about how I can position box models within the parent tags of an HTML document.

By using display: flex, flex-direction: row, justify-content: space-evenly, and flex-wrap: wrap.

By using flex I can reverse the order of positions of the box models that are made using the child tags and class.
